# Data Professional Survey Analysis

## Overview

Interactive Power BI dashboard analyzing survey data from data professionals to identify trends in salaries, job satisfaction, programming languages, work-life balance, and career entry difficulty.

## Business Questions

- Which data roles have the highest salaries?
- Which programming languages are the most used?
- How satisfied are professionals with their salary?
- Which countries have the highest participation?
- How difficult is it to enter the Data Analytics field?

## Tools

- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization

## Key Features

- Interactive filters
- KPI cards
- Salary analysis
- Country comparison
- Programming language distribution
- Work-life balance analysis
